# 新闻门户使用说明

## 📖 功能介绍

这是一个自动化的新闻门户系统，可以：
- 自动扫描 `output` 目录下的所有日期和新闻文件
- 提供清晰的日期导航和时间分类
- 使用 iframe 方式无缝嵌入新闻页面
- 支持移动端响应式设计

## 🚀 快速开始

### 方法一：使用批处理脚本（推荐）

双击运行 `start-server.bat`，然后在浏览器中访问 http://localhost:8080

### 方法二：手动启动

```bash
node server.js
```

然后在浏览器中打开：http://localhost:8080

## ✨ 自动扫描特性

- **无需手动刷新**：服务器会在每次页面加载时自动扫描 output 目录
- **实时更新**：爬虫生成新文件后，刷新浏览器即可看到最新内容
- **零配置**：无需运行额外的脚本，一切都是自动的

## 📁 文件结构

```
项目根目录/
├── index.html              # 门户主页面
├── server.js               # Node.js 服务器（自动扫描）
├── start-server.bat        # 一键启动脚本
└── output/                 # 爬虫生成的新闻目录
    ├── 2025年11月12日/
    │   └── html/
    │       └── 22时30分.html
    ├── 2025年11月11日/
    └── 2025年11月10日/
```

## 🔄 更新新闻列表

当爬虫生成新的新闻文件后，只需**刷新浏览器页面**即可自动加载最新内容！

服务器会在每次请求时实时扫描 output 目录，无需任何手动操作。

## 💡 使用技巧

- **日期导航**：点击日期可以展开/折叠该日期下的新闻列表
- **快速切换**：点击时间即可在右侧查看对应的新闻内容
- **移动端**：在手机上访问时，点击左上角菜单图标可打开侧边栏
- **最新优先**：日期和时间都按倒序排列，最新的在最上面

## 🛠️ 自定义配置

如需修改端口或其他配置，可以编辑：
- `server.js` - 修改端口号（默认8080）和扫描逻辑
- `index.html` - 修改界面样式和布局

## ❓ 常见问题

**Q: 为什么看不到新闻列表？**
A: 确保 `output` 目录下有新闻文件，并且服务器正在运行。刷新浏览器页面即可。

**Q: 如何集成到自动化流程？**
A: 无需额外集成！爬虫生成新文件后，用户刷新页面就能看到最新内容。

**Q: 可以部署到服务器吗？**
A: 可以！使用 Node.js 部署 `server.js`，或者使用 PM2 等进程管理工具保持服务运行。

## 📝 技术栈

- 纯 HTML/CSS/JavaScript（无需框架）
- Node.js HTTP 服务器（自动扫描文件系统）
- 响应式设计
- iframe 嵌入技术
- 实时文件系统扫描
